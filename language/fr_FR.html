Manuel d’installation utilisateur

App MyParcel BE Shopify

Introduction

Dans ce manuel, nous vous expliquons comment installer et configurer l’application dans votre environnement Shopify. Celle-ci vous est proposée gratuitement par MyParcel BE. Nous vous aiderons avec plaisir en cas de question sur le fonctionnement de l’application.

Toutes les fonctionnalités de l’application ont été testées dans un environnement Shopify standard.

Les prix affichés dans les fenêtres sont indiqués à titre d’exemple et peuvent différer des tarifs réels. Les prix corrects peuvent être consultés sur votre compte dans tarifs.

Sommaire

1 Installation

1.1 Via l’App Store de Shopify

  1. Accédez directement à l’application MyParcel BE via celien. Si le lien ne fonctionne pas, recherchezMyParcel BE dans l’App Store de Shopify.
  2. Cliquez sur le bouton ‘Get’ et suivez les étapes.

2 Paramétrage de l’application

2.1 Paramétrer le couplage

Après l’installation, MyParcel être ajouté dans les Apps.

Pour ajouter votre clé API et parcourir tous les autres paramètres, cliquez sur l’application MyParcel BE et ensuite sur Paramètres.

C’est ici que vous devez ajouter votre clé API. Pour utiliser l’application MyParcel BE, vous devez créer un compte MyParcel BE. Créez votre compte  sans engagement pour obtenir votre clé API.

Ensuite, choisissez vos préférences d’imprimante et la référence de l’étiquette. Vous préférez par exemple que MyParcel BE organise l’exécution du processus dans Shopify ? Cochez alors l’option Automatique.

2.2 Paramètres d'exportation standards

L’application MyParcel BE récupérera les modes d'expédition existants. Pour exporter des commandes vers MyParcel BE, il convient d’abord de coupler celles-ci, et d’éventuelles options d’expédition, à MyParcel BE. 

Si vous n’avez pas encore créé de modes d’expédition, allez dans Settings - Shipping dans votre environnement Shopify. Nous vous conseillons de créer différents modes d’expédition. Vous souhaitez proposer des points de collecte pour vos clients en Belgique ? Créez un mode d’expédition distinct avec le titre Envoi au point de collecte bpost (à sélectionner après le paiement), par exemple. Votre client saura ainsi exactement à quoi s’attendre.





Cliquez sur un mode d’expédition pour indiquer comment vous souhaitez l’exporter.  

Faites votre choix pour un type d’envoi. Vous avez aussi la possibilité d’ajouter des options d'expédition complémentaires. Bien sûr, il est également possible de ne pas lier un mode d’expédition à MyParcel BE.



2.3 Points de collecte bpost

L’application MyParcel BE offre la possibilité à vos clients situés en Belgique de choisir, au moment de leur commande,un point de collecte bpost en fonction de leur code postal et de leur numéro de rue. Voici ce que cela donne sur la page de remerciement de Shopify





Important
Avec Shopify, il est uniquement possible de proposer des options de livraison après le paiement. MyParcel BE est donc obligé de placer l’option de livraison sur la page de remerciement de Shopify. 

En plus de cette option de livraison, nous offrons également la possibilité d’envoyer un e-mail de rappel à votre client après 30 minutes, si celui-ci a oublié de choisir un point de collecte. Dans cet e-mail, votre client aura à nouveau la possibilité de choisir le point.  





Important

Si vous utilisez un mode d’expédition qui permet au client de choisir un point de collecte, nous vous recommandons d’activer cette fonctionnalité. Si votre client n’a pas choisi de point de collecte, la commande ne pourra pas être exportée vers MyParcel BE.

2.4 Envoi internationaux

Outre les commandes belges et européennes, l’application MyParcel BE permet aussi de créer des commandes pour le reste du monde (formulaire de douane compris). Spécifiez un poids standard si vous envoyez toujours le même type de produit.

Vous pouvez bien sûr également définir le poids par produit. Comme vous pouvez le voir ci-dessous, il est possible d’indiquer le poids au niveau du produit. L’application MyParcel BE inclura ces informations lors de l’exportation. 

3 Commandes

3.1 Créer des commandes et des étiquettes d'expédition

Via l’application MyParcel BE, il est possible d’exporter des commandes (par lot) et de générer des étiquettes d'expédition de deux manières. Cela peut se faire dans l’aperçu des commandes original ou sur la page principale de l’application MyParcel BE. 

Dans l’aperçu de commandes original, sélectionnez vos commandes, cliquez sur le bouton Actions,puis sur Print labels sous ‘Apps’.





Vous trouverez davantage d’informations dans l’aperçu de l’application MyParcel BE que sur la page de commande d'origine. Vous voulez savoir quel code-barres est lié à la commande ou modifier une commande en particulier en ajoutant une option d’expédition supplémentaire ? Nous vous conseillons d’utiliser l’aperçu ci-dessous.



3.2 Modifier une commande

Comme décrit dans la section précédente, vous pouvez ajuster les options d’expédition par commande. Vous souhaitez assurer l’envoi d’une commande spécifique, demander une signature pour réception ou créer plusieurs étiquettes pour une commande ? C’est possible en cliquant sur la commande dans l’aperçu de l’application MyParcel BE.



3.3 Créer une étiquette de retour

Si votre client désire renvoyer son colis, vous pouvez lui envoyer une étiquette de retour depuis la commande. Cette étiquette de retour est envoyée à l’adresse e-mail spécifiée par votre client dans la commande.





Tous les paramètres que vous avez définis dans votre environnement MyParcel BE sont utilisés pour cette fonctionnalité de retour. Ce renvoi doit toutefois remplir un certain nombre de conditions. En effet, une étiquette d’expédition doit avoir été créée et la fonctionnalité est limitée à la Belgique.
